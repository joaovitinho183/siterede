O que é uma LAN:


Uma LAN, ou Local Area Network, é um tipo de rede de computadores cobrindo uma área limitada de forma geográfica, como uma casa, escritório, ou instalação de campus. * As LANs são usadas para conectar sistemas e dispositivos quando se trata de um espaço pequeno, em que se presume que os computadores e sistemas estão em estreita proximidade de um para o outro.


O que é o MAN:

Man Metropolitan Area Network : MAN é uma rede que abrange uma área geográfica maior que uma LAN, mas menor que uma WAN. Normalmente, uma MAN cobre uma cidade ou uma imensa área metropolitana e é usada para juntar muitas LANs do interior desse raio.


O que é o WAN:

Wide Area Network destina-se a uma ampla área geográfica, por exemplo, países ou mesmo continentes. WANs conectam redes locais em locais físicos separados e, geralmente, interagem com a maior parte da rede.


O que é o PAN:

PAN é uma rede que cobre uma área muito pequena, usualmente alguns metros. É utilizada para conectar dispositivos pessoais, dispositivos móveis como smartphones, tablets e laptops, em uma distância igualmente diminuta, como insida de uma casa ou de um escritório pequeno.
